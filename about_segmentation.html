About Segmentation

The main problem that segmentation tries to solve is allowing for a bigger address space than that allowed by base and bounds. Instead of having only one base and bounds pair in the Memory Management Unit, we have one base and bounds pair per logical segment of the address space. With this concept, each segment can be placed independently in the physical address space and still map to a place easily accessible by a program in the virtual address space.
